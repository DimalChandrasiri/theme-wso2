Panels - Accordion with radio buttons

Extended panels

Add panel-extended class to the panel-group to have extended panel behaviours. This will enable you to have one or more of these extras

  • Radio buttons with checked behaviour when panel is open
  • Title and description
  • Collapsed/Expanded content status icon
