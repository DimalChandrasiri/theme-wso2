Form Validation

Applying Validations

To select the form and apply the validation add $('#form-name').validate() on the $( document ).ready() function. Also you can specify options (e.g:- maxlength, minlength, min, max etc) and custom messages inside validate method. For more details please refer jQuery Validation Plugin.

Validation Types

$('#form-validation').validate();

Range Validation

$('#form-range').validate();
