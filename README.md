# Theme WSO2

### What's included

Within the distribution download you'll find the following directories and files, 
logically grouping common assets and providing both compiled and minified variations. You'll see something like this:

```
theme-wso2/
├── public/
│   ├── ...
│   ├── bundle.css
│   ├── bundle.js
│   └── index.html
├── module/
│   ├── ...
│   ├── components/
│   ├── stylesheets/
│   ├── package.json
│   └── index.js
├── docs/
│   ├── ...
│   ├── stylesheets/
│   ├── pages/
│   └── index.jsx
├── ...
├── .babelrc
├── package.json
└── webpack.config.js
```
